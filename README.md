# Scrum Master Prompt Library

A practical collection of ready-to-use prompts for day-to-day Scrum Master scenarios:
- Retrospectives
- Daily Scrums
- Sprint Planning
- Stakeholder conversations
- Team dynamics

##Purpose
To help facilitate better conversations, handle tough situations, and drive actionable outcomes — without improvising every time.

---

##Structure

- retros/ → Retrospective facilitation prompts
- daily-scrum/ → Standup facilitation & unblocking
- sprint-planning/ → Planning & scope control
- stakeholder-management/ → Handling pressure & priorities
- team-dynamics/ → Conflict & engagement

---

##How to Use

Each file contains:
- Situation → When to use it
- Goal → What outcome you want
- Prompt → What to actually say

Keep prompts:
- Short
- Natural
- Adaptable

---

##Continuous Improvement

Add new prompts after real scenarios:
- What worked?
- What didn’t?
- What would you say differently?

---

##Tags

Use tags inside files for quick search:
`retro`, `pushback`, `conflict`, `flow`, `stakeholder`, etc.

---

##Tip

Don’t try to use everything.
Pick 1–2 prompts per situation and apply them well.
